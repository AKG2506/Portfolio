MERN | Socket.IO | TailwindCss

Code Buddy is a real-time collaborative IDE crafted for seamless peer programming and developer mentorship. Featuring multi-language support and a sleek, responsive interface built with React and TailwindCSS, it delivers an intuitive coding experience. The backend, powered by Node.js and WebSockets, was engineered to reduce latency by 35% and increase session scalability by 40%, ensuring smooth collaboration even under heavy load. Designed as a secure, production-ready MVP, Code Buddy enables live code editing, making it ideal for interviews, mentorship, and remote development workflows.

Turborepo | Next | Expressjs | Docker | Prisma

VaultPay is a secure digital wallet platform that enables top-ups, peer-to-peer transfers, and transaction analytics for over 10 mock users and 100+ transactions. Built with Next.js, PostgreSQL, and Prisma, it integrates NextAuth and JWT to provide secure authentication, reducing token validation time by 30% in backend calls. The project is containerized with Docker and optimized using Turborepo and CI/CD pipelines, improving build times by 40% and ensuring a scalable, production-grade development workflow. VaultPay also features a modern dashboard UI for seamless user interaction.

MERN | Web App | ChakraUI

Meal Monkey is a RESTful web application built using the MERN stack (MongoDB, Express, ReactJS, NodeJS), designed to streamline food ordering and delivery. The platform supports real-time order management, a dynamic menu with 50+ items, and an intuitive admin dashboard. Backend optimizations enable efficient data retrieval from MongoDB, achieving API response times under 300ms in test sessions. Deployed with a modern architecture — frontend on Vercel and backend on Render — Meal Monkey maintains 95%+ local uptime, ensuring fast and reliable access for users and admins alike.
